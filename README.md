# KnownApiForward
forward json POST api calls from IFTTT to your Known site

## IFTT

- url : url to this script
- method : POST
- contentType: json
- Body {"body":"I Push my Status to my Known site","id":"YourSecretAuth", "key":"yourSecretKey"}
